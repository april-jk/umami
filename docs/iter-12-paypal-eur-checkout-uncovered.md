# 迭代 12: PayPal EUR 订阅结账 - 未覆盖测试说明

## 未覆盖场景

- PayPal Live EUR Plan 的创建、审批、首次扣款和续费依赖 PayPal 外部服务，不能由单元测试覆盖。
- 欧洲买家是否能使用 PayPal 钱包或银行卡，取决于买家所在国家、账户状态和 PayPal 风控。
- Railway 环境变量发布与生产页面部署属于外部平台操作。

## 手动测试步骤

1. 使用 PayPal Sandbox 的欧洲买家账户打开 `/membership/upgrade`。
2. 选择 `EUR` 和任意付费套餐，确认审批页显示 EUR 金额。
3. 完成订阅后返回升级页，确认订阅被确认且权益已更新。
4. 重复使用 USD，确认美洲用户路径保持不变。

## 建议补充的测试用例

- 为德国、法国和荷兰的 Sandbox 买家账户分别完成 PayPal 钱包和银行卡订阅。
- 对 PayPal 返回的 EUR Plan 货币、金额和 `ACTIVE` 状态建立发布前只读核验。
- 启用 webhook 后验证 EUR 订阅批准事件能够在用户未返回站点时同步权益。
