# 迭代 9：会员到期时间展示未覆盖说明

## 已完成的自动化覆盖

- 使用量查询会读取订阅的当前周期结束时间，并以 ISO 时间传递给已授权成员。
- 使用量接口会原样返回会员到期时间。
- 会员页面在有到期时间时显示本地化日期，在没有到期时间时不显示该字段。

## 无法由单元测试完整覆盖的场景

### 1. 真实登录态和浏览器布局

原因：会员数据依赖真实租户订阅记录，日期呈现还依赖浏览器的区域设置和页面布局。

手动测试步骤：

1. 使用具有有效订阅且存在 `currentPeriodEnd` 的成员账号访问 `/membership`。
2. 确认“会员到期时间”显示在当前计费周期下方，日期与订阅记录一致。
3. 使用没有 `currentPeriodEnd` 的免费或永久会员账号访问同一页面，确认不显示空白的到期时间字段。
4. 切换界面语言为中文和英文，并在桌面及移动端确认标签和日期没有溢出。

建议补充的测试用例：

- 在 Playwright 登录态下覆盖真实租户接口和窄视口布局。
- 对不同浏览器时区执行视觉回归，确认 UTC 日期不会发生跨日偏移。
